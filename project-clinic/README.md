# Проект: Запись в частную клинику

## 📌 Проблема
40% пациентов звонят в регистратуру, потому что не могут найти нужного врача на сайте. 15% пациентов не приходят на приём и не отменяют его заранее, из-за чего клиника теряет до 20% выручки в месяц. Врачи не знают, кто к ним придёт, пока пациент не сел в кресло.

## 💡 Решение
Спроектировала систему онлайн-записи к врачу:
- Выбор врача по специализации и рейтингу
- Календарь со свободными слотами
- SMS-подтверждение записи
- Напоминания о приёме за 24 часа и за 2 часа
- Личный кабинет врача с историей болезни пациентов

### BPMN-схема
[bpmn-appointment.png](./bpmn-appointment.png)

**Описание:**
BPMN-схема процесса записи пациента (адаптировано на примере обработки заявки нового клиента).
Основные этапы: выбор врача → выбор времени → заполнение данных → SMS-подтверждение → запись создана.
Роли: Пациент, Система, Врач, Администратор.
