# Портфолио бизнес-аналитика

Добро пожаловать! Это портфолио демонстрирует навыки бизнес-анализа через реальные проекты и учебные работы.


## 📂 Структура портфолио

### Основные проекты (Featured Projects)

Детально проработанные проекты с полным набором артефактов: требования, моделирование процессов, проектирование БД, SQL-запросы.

#### 🛒 [Интернет-магазин](./project-ecommerce/)
**Статус:** FEATURED PROJECT (полный комплект)

Комплексная система для интернет-магазина: фильтрация товаров, корзина, оформление заказа, уведомления менеджерам.

**Артефакты:**
- 5 User Stories с контекстом и критериями приёмки
- Use Case оформления заказа (16 шагов)
- BPMN-схема процесса заказа
- ER-диаграмма базы данных
- 10 SQL-запросов для анализа данных
- Подробные описания всех артефактов

**Ключевые навыки:** User Stories, Use Cases, BPMN 2.0, ER-диаграммы, SQL, работа с требованиями


#### 🏥 [Запись в частную клинику](./project-clinic/)
**Статус:** Готов (80%)

Система онлайн-записи к врачу: выбор врача по специализации и рейтингу, SMS-подтверждение, напоминания, личный кабинет врача.

**Артефакты:**
- 3 User Stories (запись, напоминания, расписание врача)
- Use Case записи к врачу (текстовый формат)
- BPMN-схема процесса записи
- Подробное описание BPMN

**Ключевые навыки:** User Stories, Use Cases, BPMN 2.0, работа с медицинским доменом

**В разработке:** ER-диаграмма базы данных


#### 🍔 [Сервис доставки еды](./project-delivery/)
**Статус:** Готов (70%)

Система отслеживания заказов и оптимального распределения заказов для курьеров: карта в реальном времени, группировка заказов, маршрутизация.
**Артефакты:**
- 2 User Stories (отслеживание, распределение заказов)
- BPMN-схема обработки возврата денег (фокус на обработке исключений)
- Подробное описание BPMN

**Ключевые навыки:** User Stories, BPMN 2.0, работа с исключениями, логистика

**В разработке:** ER-диаграмма базы данных


### [Учебные проекты](./educational/)

Дополнительные артефакты, демонстрирующие breadth (широту) навыков — умение быстро разбираться в разных предметных областях.

**Содержимое:**
- 4 ER-диаграммы для разных доменов (отель, школа, салон красоты, фитнес-клуб)
- 1 BPMN-схема B2B-процесса (согласование нового поставщика)
- Подробные описания всех артефактов

**Что демонстрирует:**
- Умение проектировать БД для разных бизнес-моделей
- Навык моделирования сложных многоуровневых согласований
- Понимание корпоративных процессов и комплаенса


## 🛠 Инструменты и навыки

### Требования и документация
- **User Stories** с контекстом и Acceptance Criteria
- **Use Cases** (табличный и текстовый форматы)
- **Бизнес-правила** и edge cases

### Моделирование процессов
- **BPMN 2.0** (нотация)
- **StormBPMN** (инструмент)
- Основные и поддерживающие процессы
- Обработка исключений и эскалация

### Проектирование данных
- **ER-диаграммы** (сущности, связи, атрибуты)
- **Нормализация** БД
- **SQL-запросы** (SELECT, JOIN, GROUP BY, HAVING, CASE WHEN)

### Инструменты
- **Моделирование:** StormBPMN, FigJam
- **Документация:** Markdown, GitHub
- **Базы данных:** PostgreSQL

## 🎯 Что показывает это портфолио

### Глубина (основные проекты)
- Детальная проработка требований
- Полный цикл: от User Stories до SQL-запросов
- Понимание бизнес-контекста и метрик

### Ширина (учебные проекты)
- Разные домены: ecommerce, медицина, логистика, B2B
- Разные типы процессов: основные, исключения, согласования
- Разные бизнес-модели: услуги, продажи, подписки

### Профессиональный подход
- Каждый артефакт имеет подробное описание
- Честное указание статуса (готово / в разработке)
- Фокус на бизнес-ценности, а не только на технике
