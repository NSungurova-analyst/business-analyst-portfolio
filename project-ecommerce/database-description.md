## 🛒 БАЗА ДАННЫХ ДЛЯ ОНЛАЙН-МАГАЗИНА

### Описание
База данных для интернет-магазина. Система управляет каталогом товаров, заказами покупателей, оплатой, доставкой и отзывами о товарах.

### Количество сущностей
10 таблиц

### Основные сущности
- **Покупатель** — клиенты магазина (включая email, пароль, подтверждение)
- **Товар** — продукция магазина
- **Категория товара** — классификация товаров
- **Заказ** — покупки клиентов (структурный центр)
- **Состав заказа** — товары в заказе (промежуточная таблица)
- **Оплата** — платежи по заказам
- **Доставка** — способы и адреса доставки
- **Отзыв** — оценки товаров покупателями
- **Сотрудник** — персонал (курьеры, операторы)
- **Справочник должностей**

### Ключевые решения
- Состав заказа вынесен в отдельную таблицу для связи M:N между заказом и товаром (один заказ может содержать много товаров)
- Регистрация объединена с покупателем (email, пароль, подтверждение)
- Оплата и доставка вынесены в отдельные таблицы для учёта нескольких платежей и адресов
- Паспортные данные сделаны опциональными (только для международных заказов)
- Структурный центр — Заказ (связывает покупателя, товары, оплату, доставку)

### Связи
- 1:M: Покупатель → Заказ
- M:N: Заказ ↔ Товар (через Состав заказа)
- 1:M: Заказ → Оплата
- 1:M: Заказ → Доставка
- M:N: Покупатель ↔ Товар (через Отзыв)
- 1:M: Категория товара → Товар

### Сложности
- Связь M:N между заказом и товаром через состав заказа с учётом количества и цены на момент покупки
- Отзыв связан с заказом для подтверждения покупки товара
- Несколько способов оплаты и доставки для одного заказа

### Возможности развития системы
- Маркетинговый модуль: промокоды, акции, персональные скидки
- Складской учет: наличие товаров на складе, автоматические уведомления о заканчивающихся товарах
- Аналитика: история изменения цен, отчеты по продажам
- Интеграции: подключение платежных систем и служб доставки
