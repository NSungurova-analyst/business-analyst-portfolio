# Проект: Интернет-магазин

## 📌 Проблема
По данным аналитики, 70% пользователей уходят со страницы каталога, не найдя подходящий товар. 30% бросают оформление заказа на этапе выбора доставки. Менеджеры не видят новые заказы в реальном времени, что увеличивает время обработки.

## 💡 Решение
Спроектировала комплексную систему для интернет-магазина:
- Фильтрация и сортировка товаров по цене и рейтингу
- Удобная корзина с сохранением на 30 дней
- Прозрачный процесс оформления заказа с выбором доставки
- Уведомления для менеджеров о новых заказах
