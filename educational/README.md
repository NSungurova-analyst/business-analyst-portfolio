# Учебные проекты и дополнительные артефакты

Эта папка содержит учебные работы и дополнительные артефакты, которые не вошли в основные проекты, но демонстрируют широту навыков бизнес-анализа.

## 📦 Содержимое

### Базы данных (ER-диаграммы)

Коллекция ER-диаграмм для различных доменов, демонстрирующая умение проектировать структуры данных для разных бизнес-моделей.

**Файлы:**
- [database-hotel.png](./database-hotel.png) — База данных отеля
- [database-school.png](./database-school.png) — База данных школы
- [database-salon.png](./database-salon.png) — База данных салона красоты
- [database-fitness.png](./database-fitness.png) — База данных фитнес-клуба

**Подробные описания:**
- [database-collection.md](./database-collection.md) — Описания всех 4 БД с указанием сущностей, связей и бизнес-правил

**Что демонстрирует:**
- ✅ Умение проектировать БД для разных доменов (гостиницы, образование, услуги, спорт)
- ✅ Понимание различных бизнес-моделей и их отражения в данных
- ✅ Навык работы с разными типами связей (один-ко-многим, многие-ко-многим)
- ✅ Знание нормализации и оптимизации структуры данных


### BPMN-схемы

Дополнительные BPMN-схемы, демонстрирующие умение моделировать сложные B2B-процессы.

**Файлы:**
- [bpmn-supplier.png](./bpmn-supplier.png) — Согласование нового поставщика

**Подробное описание:**
- [bpmn-supplier-description.md](./bpmn-supplier-description.md) — Детальное описание процесса с ролями, шлюзами, бизнес-правилами

**Что демонстрирует:**
- ✅ Умение моделировать сложные многоуровневые согласования
- ✅ Понимание корпоративных процессов и разделения ответственности
- ✅ Навык работы с комплаенсом и проверкой контрагентов
- ✅ Знание финансовых критериев оценки поставщиков
- ✅ Умение проектировать эскалацию и обработку разногласий

---

## 🎯 Зачем нужна эта папка

Основные проекты (`project-ecommerce`, `project-clinic`, `project-delivery`) показывают **глубину** навыков — детально проработанные артефакты для конкретных доменов.

Эта папка показывает **широту** навыков — умение быстро разбираться в разных предметных областях и создавать артефакты для них.

**Итого:**
- 4 ER-диаграммы для разных доменов
- 1 сложная BPMN-схема (B2B-процесс)
- Подробные описания всех артефактов

Это демонстрирует **универсальность** и готовность работать с разными бизнес-задачами.
